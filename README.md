# slm-agentic-software-engineering
Research project on improving Small Language Models using Agentic Execution Traces for Software Engineering tasks.

# SLM Agentic Software Engineering

## Overview

This project investigates whether a lightweight Small Language Model (SLM) can improve software engineering performance through post-training on agentic execution traces.

The project compares:

- Baseline Local SLM
- Multi-Agent Workflow
- Fine-Tuned SLM

using SWE-Bench Lite as the evaluation benchmark.
