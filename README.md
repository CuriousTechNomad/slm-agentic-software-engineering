# SLM Agentic Software Engineering

## Overview

This project investigates whether a lightweight Small Language Model (SLM) can improve software engineering performance by learning from Agentic Execution Traces.

The project follows three stages:

1. Baseline Local SLM
2. Multi-Agent Workflow
3. Fine-tuned Local SLM

Evaluation will be performed using SWE-Bench Lite.

---

## Objectives

- Evaluate a baseline SLM on SWE-Bench Lite.
- Build an agentic workflow for software engineering tasks.
- Generate agentic execution traces.
- Fine-tune the SLM using generated traces.
- Compare performance before and after training.

---

## Tech Stack

- Python
- Google Colab
- Hugging Face Transformers
- LangGraph (or equivalent)
- PEFT
- TRL
- FAISS
- SWE-Bench Lite
